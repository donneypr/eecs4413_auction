----------------------------
Initial Setup
----------------------------

Before starting, make sure you have installed:

    Docker Desktop: 

        Download: https://www.docker.com/products/docker-desktop/
    
    Python 3.10+

    Git: 

        Download: https://www.docker.com/products/docker-desktop/


----------------------------
Download
----------------------------

Either:

Clone the Repository:

    On Terminal/Command Prompt:

    git clone https://github.com/donneypr/eecs4413_auction.git


Or:

Download the File:

    1. Go to https://github.com/donneypr/eecs4413_auction

    2. Press "<> Code"

    3. Press "Download ZIP"

    4. Save the folder anywhere

----------------------------
.env and .env.db Files
----------------------------

Copy and paste the .env and .env.db files sent separately into the "eecs4413_auction" folder.

-----------------------------
Docker
-----------------------------

On VSCode terminal:

    cd <path>/eecs4413_auction
    docker compose up --build

-----------------------------
Frontend and Backend
-----------------------------

1. Go to a web browser

2. On URL/Search bar, type:

    For frontend:

        http://localhost:3000

    For backend:

        http://localhost:8000
